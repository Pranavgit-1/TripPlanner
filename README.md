 TripPlanner - Interactive Travel Booking Website

Features
- Simple Calendar Date Picker
- Flight/Train/Bus Selection  
- Real-time Schedule Options
- Ticket Download (PDF format)
- Fully Responsive Design

Tech Stack

How to Use
1. Click **PLANNING** button
2. Fill trip details (people, from/to, **date**)
3. Choose transport mode
4. Select preferred timing
5. **Download your ticket!**
